## 2. NumPy: Numerical Computing

NumPy’s core feature is the ndarray (N-dimensional array), which allows fast, memory-efficient operations on large datasets.

- NumPy arrays are more efficient than Python lists for numerical data.
- Supports element-wise operations, aggregation (sum, mean, std), and advanced indexing.
- Used for image processing, signal analysis, simulations, and any task requiring fast numerical computation.
- Essential for preparing data before using machine learning models.

Array creation and manipulation
Indexing, slicing, broadcasting
Statistical analysis (mean, std, percentiles)
Random sampling and simulation
Matrix operations (dot product, covariance, correlation)
Data normalization and masking
